Backend-часть интернет-магазина по продаже котлов и отопительного оборудования. Архитектура построена на модульной структуре NestJS с использованием Sequelize ORM для работы с MySQL базой данных.
Проект создан в учебных целях и представляет собой pet-проект, разрабатываемый по урокам с YouTube.

Функционал приложения:

- Аутентификация и авторизация пользователей;
- Обработка заказов и корзины покупателя;
- Управление каталогом товаров;
- Swagger-документация;
- Валидация входящих данных.

На данный момент реализован базовый функционал, но работа над проектом продолжается. Планируется:

- Расширение функциональности каталога товаров;
- Добавление системы скидок и промокодов;
- Добавление административной панели для управления контентом.
